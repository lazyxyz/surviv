Welcome to SurvivFun

SurvivFun is a play-and-earn game built entirely on the blockchain.

- All game items are NFTs stored securely on the blockchain.
- All items are created directly on-chain for transparency.

How to Get Game Items

Open crates to get game items. Each crate requires one key to unlock.

How to Get Crates and Keys

You can get crates and keys in two ways:

1. Buy from the Shop
The quickest way to grab crates and keys is to buy them from the in-game shop.
